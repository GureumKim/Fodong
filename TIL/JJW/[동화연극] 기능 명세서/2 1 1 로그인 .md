**BACKEND :: SESSION 방식**

1. **`DB`**에서 사용자 확인
2. **`세션 저장소`**에 회원 정보 세션 생성
3. 세션 저장소에서 생성된 `SessionID` 을 클라이언트에 전달

**FRONT**

1. 서버로 로그인 요청을 보낸다

→ FORM 형식? GET요청? JWT?>

(일단 명세에는 local storage 사용이긴 한데 바꿀 수 있음)

**BACKEND :: JWT 방식**

1. `DB`에서 사용자 확인
2. Access Token(JWT) 발급
3. 클라이언트에 응답
    - `Access Token` 전달

**FRONT**

1. 서버로 로그인 요청 보낸다

if 성공시

1. 메인 페이지로 리다이렉트 한다.

elif 실패시
